# 專案指引

## 專案目錄結構

 - Django Templates ： 《Project》/Templates/
 - 靜態檔案： 《Project》/static/

## 建置平台

### 安裝 Bootstrap

 ```
 yarn add bootstrap
 ```

 ```
 yarn global add bootstrap
 la ~/.config/yarn/global/node_modules/bootstrap/
 ```

## 功能

### 使用者管理

 - 登入： /account/login/
 - 登出： /account/logout/
 - 註冊： /account/registration/
 - 變更密碼： /account/password_change/
 - 重設密碼： /account/password_reset/
